# Gestione Turni

PWA mobile-first per la gestione dei turni settimanali. Dati sincronizzati su Firebase Firestore.

## Avvio in locale

```bash
python3 -m http.server 8000
```

Poi apri il browser su: [http://localhost:8000](http://localhost:8000)

## Deploy

Il progetto è pubblicato su GitHub Pages:
**https://baccadiginepro.github.io/weekly-shifts**

## Stack

- HTML / CSS / JavaScript vanilla (nessun framework)
- Firebase Firestore (sync cloud)
- PWA installabile su Android tramite Chrome ("Aggiungi a schermata Home")
